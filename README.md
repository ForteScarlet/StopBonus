## 使用

**数据保存位置**

Windows 上数据默认储存在 `$USER_HOME\AppData\Local\StopBonus`

**如何使用**

奖励完了填个表。

## 更新

比较建议先卸载旧的，再安装新的。如果数据被保存在安装目录内，记得先备份。
别问，compose打包出来的安装包似乎对更新安装不太友好。
